# Nuxt blog

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

```

## json-server

Start the development server on `http://localhost:3030`:

```bash
# npm
npx json-server db/db.json --port 3030

```
